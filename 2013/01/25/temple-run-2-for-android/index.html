Temple Run 2 for Android 已在 Google Play 发布，但不支持中国区域账号下载。

在上周发布 iOS 版本的 Temple Run 2 后，Android 版本如期而至，目前已登陆 Google Play，像青小蛙这样肯折腾的家伙，把 Android 设备 root 后，模拟美国市场就下载到了…

更折腾的同学，会从 Google Play 上下载 APK 文件，然后分享给大家…
